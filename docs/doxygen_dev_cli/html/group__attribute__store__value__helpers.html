Detailed Description

Helper functions to perform actions on the attribute store.

This module contains various helper function to ease the access of the attribute store.

◆ attribute_store_walk_tree()

sl_status_t attribute_store_walk_tree ( attribute_store_node_t  top,
sl_status_t(*)(attribute_store_node_t, int)  func,
int  depth 
)

Visit all nodes in the tree.

This function visits all tree nodes going from the top and down. It will continue to the node children if the status returned by the function is SL_STATUS_OK. If the function returns SL_STATUS_ABORT, no more nodes are visited. Otherwise, siblings will be visited.

Parameters
topFirst node to be visited
funcfunction to execute
depthThe current depth of the top node.
