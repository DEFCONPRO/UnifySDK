class  CommandTemplate
 This class is a generic class for implementing a command into the command client interface tool. The class has a void function command_to_execute which should be populated by each command src file, thus the command handler knows for each object which function to execute when a specific command is requested in the CLI. Detailed Description

CLI command runner and management.

Contiki process which manages the user commands and exits the application when a command broadcasts COMMANDS_DONE.
