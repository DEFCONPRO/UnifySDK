Detailed Description

This class is a generic class for implementing a command into the command client interface tool. The class has a void function command_to_execute which should be populated by each command src file, thus the command handler knows for each object which function to execute when a specific command is requested in the CLI.

Parameters

infohelping information for using the command
arg_helpusage for aguments help
