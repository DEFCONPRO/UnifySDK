MqttGroupHandler handles subscription messages related to UIC groups. It subscribes to listen to the correct topics and is to be passed to the mosquitto client

pub fn new(client: T) -> Result<Self, sl_status_t>

crate new handler to be passed into a MqttClient. The native mqtt client is passed as reference counted. Reference counted objects can be shared and borrowed over multiple instances. Acquirring mutablity is only possible when the ref count == 1. since this rc is shared between MqttClient and MqttGroupHandler, reference count can never be == 1. hence mutable cannot be accuired for mqtt client.
