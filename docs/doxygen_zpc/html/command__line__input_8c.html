uint16_t read_command_line ( char *  user_buffer,
uint16_t  user_buffer_length,
uint16_t  timeout 
)

Read data from the command line (stdin) and copies it in the user buffer. It strips the line return characters from the input.

Parameters
user_bufferbuffer for the data to be copied in, if any
user_buffer_lengthLength of the user buffer. If the user_buffer is too small to contain the input data, only the fitting size of bytes will be copied
timeoutMaximum time to wait in ms for input.
Returns
the length of the input data copied into the user buffer. Will be 0 if no data was copied or if timed out waiting for command line input.
