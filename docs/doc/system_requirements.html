System Requirements

The Unify Host SDK is at the moment only implemented for the Linux OS. The table below shows the resource usage of the individual components measured on armhf release binaries. No debug symbols included. RAM figures are indicative for applications without any data added (i.e. initial memory usage). The measurement used for the RAM figures is the programs Resident Set Size (RSS) measured by ps or smem. Bear in mind that the RAM measurement for each application also includes memory used by shared libraries, thereby the actual memory consumed when running multiple Unify applications that use the same shared library (libuic and various system libraries) is lower than the sum of this table.

  • Release binaries are compiled with -O2 -s

  • RAM size = Resident Set Size (RSS)
