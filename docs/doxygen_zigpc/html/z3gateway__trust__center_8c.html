◆ z3gatewayZigbeeKeyEstablishmentCallback()

void z3gatewayZigbeeKeyEstablishmentCallback ( const EmberEUI64  partner,
EmberKeyStatus  status 
)

Z3Gateway handler for Zigbee Key Establishment.

A callback to the application to notify it of the status of the request for a Link Key. The status will cover stages from responding to a link key request, potential failure paths, to final link key verification. The link key verification stage is used by Z3Gateway to indicate the end of successful device addition to the network any notify the onTrustCenterDeviceJoinComplete callback for that particular device.

Parameters
partnerpartner The IEEE address of the partner device. Or all zeros if the Key establishment failed. Ver.: always
statusThe status of the key establishment. Ver.: always
