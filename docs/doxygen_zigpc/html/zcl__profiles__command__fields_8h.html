Parse and build the ZCL frame using the command fields passed in. This function will parse the informing MQTT message for ZCL commands and extract any known command fields that is associated with the command (ex. the fields: Level, TransitionTime, OptionsMask, OptionsOverride associated with the Level cluster command 'MoveToLevel').

Parameters
